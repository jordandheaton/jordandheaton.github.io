**Goal:** Ship the leaf→Earth half of the Universe Scroller as a working scroll page: 9 chained anchor stills, 8 seam-pinned 1080p clips, a windowed-loading canvas scrub engine with a scientific HUD.

**Architecture:** Anchor ladder generated by chained image-to-image (each stop references the previous), transition clips pinned at both ends (`start_image`+`end_image`), all finals stitched with 0.25s crossfades into one global frame sequence scrubbed on `<canvas>` with an index-windowed loader (network + decode bounded).

**Tech Stack:** Higgsfield MCP (`nano_banana_pro`, `seedance_2_0_mini`, `seedance_2_0`), ffmpeg (winget path, NOT on PATH), vanilla JS (no build step, matches repo), Python http.server for local verify.

### Task 5: Scroll engine page

**Files:**
- Create: `index.html`, `scroller.css`, `scroller.js` (in `Universe scroller/`)

**Interfaces:**
- Consumes: `frames/manifest.json` (Task 4 schema).
- Produces: `window.__uni` debug hook `{idx, target, loadedCount, windowRange}` used by Task 6 verification.

Engine requirements (adapt the working `demo/index.html`, which is the reference implementation for scrub/HUD/hint/loader):

1. **Windowed loading replaces all-preload.** `AHEAD=220, BEHIND=120` frames around current index: `ensureWindow(idx)` creates `Image` objects for missing indices in `[idx-BEHIND, idx+AHEAD]`, evicts (`img.src='';` delete from Map) outside `[idx-2*BEHIND, idx+2*AHEAD]`. Draw holds last-good frame if target not yet loaded (never blank).
2. **Boot:** fetch manifest → load first 60 frames before dismissing loader → start rAF loop.
3. Scrub identical to demo: `target = scrollY/PX_PER_FRAME` (PX_PER_FRAME=18), `playhead += (target-playhead)*0.22`, cover-draw, dpr≤2.
4. HUD identical to demo but data-driven from manifest chapters: `dist = 10^(a0+(a1-a0)*t)`; format: `<1 m` → cm, `<1e6` → km with 1 decimal, `≥1e6` → localized km. Label = chapter label.
5. `prefers-reduced-motion`: skip engine; render a static vertical list of 9 anchor stills (`frames/frame-0001.jpg` + each chapter's start frame) with labels.
6. Page chrome: dark `#04060c`, mono HUD (reuse demo CSS), title `UNIVERSE SCROLLER`, small `← back to portfolio` link to `../index.html`, cache-bust query `?v=1` on css/js.
