# jordandheaton.github.io

Personal portfolio of Jordan Heaton — Information Systems student at Brigham Young University.

**Live site:** https://jordandheaton.github.io

## What's here

- **3D scroll hub** (`index.html`, `portfolio-3d.css`, `portfolio-3d.js`) — scroll-driven homepage with a walk-in hero video and a laptop fly-through into the project grid. Vanilla JS + GSAP ScrollTrigger + Lenis.
- **[BYU Scholarship Matcher](BYU%20Scholarship%20Matcher/)** — matches students to BYU scholarships from their talents and skills.
- **[Liquid Simulator](Liquid%20Simulator/)** — real-time canvas fluid & particle physics simulation.
- **[IS 201 Web Project](IS%20201%20Web%20Project/)** — interactive résumé site, the BYU Target Chaser arcade game, and The Art of Ballroom feature page.
- **[Process Analysis Project](Process%20Analysis%20Project/)** — a measured case study of a real daily workflow.
- **[myplanBYU](myplanBYU/)** — MyMAP-style degree planner built on real BYU catalog data, with a Python scraper pipeline (`myplanBYU/scraper/`).

## Notes

- Draft 3D renders (`assets-3d/*.mp4` except the hero) and full-res photo originals (`Images/originals/`) are kept local and gitignored — the repo tracks only what the live site uses.
- Everything is static; no build step. Push to `main` to deploy via GitHub Pages.
